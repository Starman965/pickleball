Tournament Manager Instructions

How to run brackets, round robin, and partner rotation

Home Tournament Manager Instructions

Quick Start

  1. Open the Tournament Manager.
  2. Create a tournament (name + type + format).
  3. Add players (or teams for doubles).
  4. Generate the bracket / schedule.
  5. Enter scores as matches finish.

Create a Tournament

  • Name: any label you want (e.g. “Saturday Ladder”).
  • Type: Singles, Doubles, or Partner Rotation.
  • Format: Single Elimination or Round Robin (disabled for Partner Rotation).

Add Players (and Teams)

  • Add players: type a name and press Enter or tap Add.
  • Doubles: create teams using Random Teams or Balanced Teams (by rating).
  • Ratings: optional—use them to balance doubles teams.

Generate Brackets / Schedules

  • Single Elimination: generates a bracket and advances winners.
  • Round Robin: generates matchups for everyone vs everyone, plus standings.
  • Partner Rotation: generates round schedules and keeps standings.

Enter Scores

  • Tap Enter Score / Edit Score, input both scores, then Save.
  • Winners advance automatically in Single Elimination.

Saving & Privacy

  • Your tournaments are stored locally in your browser (localStorage).
  • Clearing browser data may remove saved tournaments.
